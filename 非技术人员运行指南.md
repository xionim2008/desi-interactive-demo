# 非技术人员运行指南

适用内容：`DESI 企业展厅多媒体内容设计工作流`互动屏  
适用对象：设计师、项目经理、现场执行人员、不会代码的操作人员

## 1. 这套文件是什么

这不是普通图片，也不是 PPT。

它是一套可以在电脑浏览器里打开的 `互动网页原型`，适合放在展厅触摸屏或多媒体屏幕上运行。

你可以把它理解成：

- 像网页一样打开
- 像小程序一样可以点击
- 像展厅屏幕内容一样可以全屏展示

## 2. 需要拷贝哪些文件

不要只拷贝一个文件。

需要把整个文件夹里的这些文件一起拷贝到多媒体设备或设备背后的电脑里：

- `index.html`
- `styles.css`
- `app.js`
- `desilogo.jpg`

如果要同时给执行公司看说明，也可以一起拷贝：

- `现场执行实施说明.md`
- `非技术人员运行指南.md`

最简单的做法：

直接把整个文件夹 `2026-04-19-new-chat` 拷贝过去。

## 3. 放到哪里

建议放到多媒体设备连接的 Windows 主机里，比如：

- `D:\DESI互动屏\`
- `C:\DESI互动屏\`
- 桌面上的 `DESI互动屏` 文件夹

建议不要放在很深的目录里，也不要放在微信、QQ、下载临时目录里。

推荐路径示例：

```text
D:\DESI互动屏\
```

文件夹里应该能看到：

```text
index.html
styles.css
app.js
desilogo.jpg
```

## 4. 怎么打开

进入文件夹后，双击：

```text
index.html
```

电脑会用浏览器打开它。

推荐使用：

- Chrome 浏览器
- Edge 浏览器

不建议使用很旧的浏览器。

## 5. 怎么全屏

打开后，按键盘上的：

```text
F11
```

页面就会进入全屏模式。

如果要退出全屏，再按一次：

```text
F11
```

如果现场使用触摸屏，建议开机后直接全屏，不要露出浏览器地址栏。

## 6. 现场怎么操作

打开后会先看到待机页。

操作方式：

1. 点击 `点击进入`
2. 进入首页总览
3. 点击任意流程阶段查看详细内容
4. 点击 `返回首页` 回到总览
5. 点击顶部按钮切换 `工作流`、`交付模板`、`落地实施`

如果长时间没有操作，页面会自动回到待机状态。

## 7. 怎么设置开机自动运行

如果只是临时演示，可以不用设置开机自动运行。

如果要长期放在展厅里，建议设置开机自动打开。

### 简单做法

1. 在电脑上找到 `index.html`
2. 右键点击它
3. 选择 `打开方式`
4. 选择 `Chrome` 或 `Edge`
5. 打开后按 `F11` 全屏

这个方式最简单，但每次开机后需要人工打开。

### 更适合现场的做法

让现场执行公司或电脑技术人员设置：

- Windows 开机自动登录
- 开机自动打开浏览器
- 浏览器自动打开 `index.html`
- 浏览器自动进入全屏

你可以直接把下面这句话发给执行公司：

```text
请将 DESI 互动屏文件放到本地主机固定目录，并设置 Windows 开机自动启动 Chrome 或 Edge，全屏打开本地 index.html 文件。
```

## 8. 给执行公司的开机启动说明

执行公司需要完成：

- 将整套文件放到固定目录
- 设置电脑开机自动登录
- 设置浏览器开机自动打开本地 `index.html`
- 设置浏览器全屏或 kiosk 模式
- 测试断电重启后是否能自动恢复

推荐运行方式：

```text
Chrome / Edge 全屏或 Kiosk 模式运行本地 index.html
```

## 9. 什么是 Kiosk 模式

Kiosk 模式可以理解成“展厅专用全屏模式”。

它的好处是：

- 不显示浏览器地址栏
- 不显示多余按钮
- 更像正式展厅程序
- 不容易被误操作退出

这个设置一般交给执行公司或技术人员完成。

## 10. 如果现场打不开怎么办

先检查这几件事：

1. 文件有没有拷贝完整
2. `index.html`、`styles.css`、`app.js`、`desilogo.jpg` 是否在同一个文件夹
3. 是否用了 Chrome 或 Edge 打开
4. logo 是否显示
5. 页面能不能点击
6. 是否被杀毒软件或系统权限拦截

最常见的问题是：

只拷贝了 `index.html`，没有拷贝其他文件。

如果只拷贝 `index.html`，页面可能会：

- 没有样式
- 不能点击
- logo 丢失
- 显示不完整

## 11. 可以只发一个链接吗

可以，但不推荐作为展厅长期运行方式。

如果把它放到服务器上，就可以生成一个在线链接，设备打开网址即可。

但展厅现场更推荐本地运行，因为：

- 不依赖网络
- 更稳定
- 加载更快
- 不容易因为网络断开导致黑屏

所以建议：

- 临时远程预览：可以用在线链接
- 正式展厅运行：建议本地文件运行

## 12. 现场交付前检查清单

交付前请检查：

- 页面能正常打开
- 页面能全屏
- 触摸点击正常
- logo 正常显示
- 首页总览正常显示
- 各阶段可以点击
- 自动轮播正常
- 长时间不操作能回到待机页
- 断电重启后能恢复运行
- 屏幕比例没有变形

## 13. 最简单的一句话

如果你要告诉现场人员怎么做，可以这样说：

```text
请把整个互动屏文件夹拷贝到多媒体主机里，双击 index.html 用 Chrome 或 Edge 打开，然后按 F11 全屏；正式运行时请设置开机自动全屏打开这个 index.html。
```
