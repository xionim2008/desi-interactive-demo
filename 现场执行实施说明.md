# DESI 正式展厅版互动屏实施说明

适用项目：`企业展厅多媒体内容设计工作流`互动展示屏  
当前演示入口：[index.html](/Users/desi/Documents/Codex/2026-04-19-new-chat/index.html)

## 1. 项目目标

本项目用于 DESI 公司展墙中的触摸互动屏展示，内容主题为“企业展厅多媒体内容设计工作流”。  
目标是以正式品牌化的界面，向客户与来访者展示 DESI 在企业展厅数字内容中的方法体系、交付能力与落地流程。

## 2. 推荐硬件配置

### 屏幕

- 尺寸建议：`55 英寸` 或 `65 英寸`
- 比例建议：`16:9 横屏`
- 分辨率建议：`3840 x 2160`
- 类型建议：`电容触摸一体机`
- 触控建议：`10 点触控`

### 主机

- 系统建议：`Windows 10/11 Pro`
- 形式建议：`迷你主机` 或 `工控主机`
- 输出接口：`HDMI 2.0` 或以上
- 存储建议：`SSD`
- 网络建议：`有线网络优先`

### 现场配套

- 壁挂支架
- 隐藏布线
- 独立供电
- 检修口
- 可选中控或定时开关
- 可选人体感应器

## 3. 运行方式

推荐使用 `HTML/CSS/JavaScript` 本地运行方案。

当前文件组成：

- [index.html](/Users/desi/Documents/Codex/2026-04-19-new-chat/index.html)：页面结构
- [styles.css](/Users/desi/Documents/Codex/2026-04-19-new-chat/styles.css)：视觉样式
- [app.js](/Users/desi/Documents/Codex/2026-04-19-new-chat/app.js)：交互逻辑
- [desilogo.jpg](/Users/desi/Documents/Codex/2026-04-19-new-chat/desilogo.jpg)：品牌 logo
- [非技术人员运行指南.md](/Users/desi/Documents/Codex/2026-04-19-new-chat/非技术人员运行指南.md)：给非技术人员的运行步骤

现场推荐两种运行方式：

1. 浏览器全屏运行
- Windows 开机后自动启动 Chrome
- 自动打开本地 `index.html`
- 浏览器进入全屏模式

2. 封装为桌面应用
- 使用 Electron 或浏览器壳封装
- 开机自动启动应用
- 禁止地址栏和系统界面露出

正式项目更推荐第 2 种，因为更像完整设备界面。

## 4. 当前已实现的交互逻辑

- 待机页
- 点击进入
- 首页总览
- 8 个阶段切换
- 交付模板页
- 落地实施页
- 自动轮播
- 返回首页
- 无操作回待机

当前逻辑参数：

- 自动轮播间隔：`9 秒`
- 无操作待机时间：`45 秒`

如现场节奏需要，可继续调整。

## 5. 开机自启建议

### 方案 A：Windows 启动项

- 设置系统自动登录
- 将 Chrome 快捷方式加入启动文件夹
- 启动参数指向本地页面并进入全屏

### 方案 B：Shell 启动器

- 使用专用启动器先启动应用
- 自动隐藏系统桌面
- 异常退出后自动重启应用

正式展厅项目建议增加：

- 断电重启自动恢复
- 浏览器崩溃自动重开
- 鼠标隐藏
- 系统更新关闭或延后

## 6. 现场安装流程

1. 确认墙面结构和屏幕安装高度
2. 确认供电、网线、检修口位置
3. 安装触摸屏与主机
4. 配置系统开机自启
5. 拷贝本地展示文件
6. 全屏运行并联调
7. 测试待机、轮播、触摸、返回首页
8. 完成收口与运维交接

## 8. 联调测试清单

- 开机后是否自动进入展示程序
- 是否全屏显示
- 触摸是否准确
- 页面切换是否流畅
- 待机页是否正常出现
- 点击进入是否正常
- 自动轮播是否正常
- 45 秒无操作是否回待机
- 字体与图像是否清晰
- 断电后重启是否恢复

## 9. 对执行公司的说明

本项目视觉方向要求：

- 白色主调
- 黑灰文字
- 无圆角
- 简洁、克制、具有设计感
- 保持 DESI 品牌气质

执行时不要擅自增加：

- 发光特效
- 过多彩色渐变
- 圆角卡片
- 复杂阴影
- 过于花哨的切换动画
